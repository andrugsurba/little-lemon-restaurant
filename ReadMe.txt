# Little Lemon Restaurant API

## Project Overview
This project is a back-end application for the Little Lemon Restaurant. It uses Django REST Framework to manage table bookings and menu options.

## Setup and Installation
### Requirements
- Python 3.x
- Django
- Django REST Framework
- MySQL

### Installation Steps
1. Clone the repository: `git clone [Your Repository URL]`
2. Navigate to the project directory: `cd [Project Directory]`
3. Install dependencies: `pip install -r requirements.txt`
4. Set up your MySQL database (details on setting up MySQL database).

## Configuration
- Configure your database settings in `settings.py`.
- Set environment variables for sensitive information.

## Running the Project
Run the server using `python manage.py runserver`.

## API Documentation
### Endpoints
- `/api/bookings/`: Manage table bookings.
- `/api/menu/`: Access and update menu items.
- `/api/registration/`: User registration.
